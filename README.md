# Ultima-Hallowed-Realms-Map-Tools

Map Tools For Ultima Hallowed Realms
---------------------------------------------------
1.) Use the included installer file first!

2). Open Cedserver.xml in any txt editor of your choice. Set it up for the Map, Staidx, and Statics file then set up the map size as shown below!

    <Map>ClientFiles/map1.mul</Map>
    <StaIdx>ClientFiles/staidx1.mul</StaIdx>
    <Statics>ClientFiles/statics1.mul</Statics>
    <Width>896</Width>
    <Height>512</Height>

3.) Now run the Start Server.

4.) Now run the Start Client.

5.) Everything is set up for local host so no need to change it.
