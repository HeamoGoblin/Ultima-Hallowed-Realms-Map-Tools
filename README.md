# Ultima-Hallowed-Realms-Map-Tools

Map Tools For Ultima Hallowed Realms
---------------------------------------------------
1.) You will have to move the client Data Files into a folder you make called ClientFiles!

2.) Use the included installer file!

3.) Open Cedserver.xml in any txt editor of your choice. Set it up for the Map, Staidx, and Statics file then set up the map size as shown below!

    <Map>ClientFiles/map1.mul</Map>
    <StaIdx>ClientFiles/staidx1.mul</StaIdx>
    <Statics>ClientFiles/statics1.mul</Statics>
    <Width>896</Width>
    <Height>512</Height>

4.) Now run the Start Server.

5.) Now run the Start Client.

6.) Everything is set up for local host so no need to change it.

Account: admin

Password: admin

Now you are ready to start making map changes to your hearts content!
